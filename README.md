### Project Overview
Analysis of the US road network dataset
using Neo4j graph database, Python, NetworkX,
and Plotly.

### Dataset
- 87,575 intersections (nodes)
- 121,961 roads (edges)
- Average degree: 2.8

### Files
- loading_script.ipynb → Python analysis notebook
- usa.txt → US road network dataset

### Tools Used
- Neo4j Community Edition 5.26.0
- Python 3.12
- NetworkX
- Plotly
